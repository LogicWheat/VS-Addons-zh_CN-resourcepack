# 项目简介
此资源包为 [瓦尔基里天空](https://modrinth.com/mod/valkyrien-skies) 及其大部分附属模组添加了简体中文翻译。<br>
资源包内也包含一些其他模组的翻译。<br>
**所有可提交至对应仓库的翻译文件均已提交至对应的官方仓库！**<br>
[瓦尔基里天空-MC百科界面](https://www.mcmod.cn/class/1528.html)
